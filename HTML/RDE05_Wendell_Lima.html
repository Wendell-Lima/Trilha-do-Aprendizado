O que são os Hiperlinks?

É uma referência dentro de um documento em hipertexto para outras partes do documento, outros documentos, arquivos ou endereços eletrônicos.
Os links podem ser remotos, internos, locais, com endereço de correio ou arquivos.

Como utilizar Hiperlinks?

A marcação de hiperlinks acontece utilizando a tag <a> e a propriedade href que faz referência ao conteúdo desejado.
a = âncora "anchor".

Sintaxe Geral:

<a href="endereço"> Conteúdo Referenciado Clicável </a>

Para abrir em uma outra aba, use: (target="_blank")
<a href="endereço" target="_blank"> Conteúdo Referenciado Clicável </a>

Para referenciar o link, use: (rel="external")
<a href="endereço" target="_blank" rel="external"> Conteúdo Referenciado Clicável </a>

Para dizer qual o idioma do site e abrir uma aba de tradutor, use: (hreflang="en,pt-br,etc")
<a href="endereço" target="_blank" rel="external" hreflang="en"> Conteúdo Referenciado Clicável </a>

Hiperlinks externos ou remotos

São links que referenciam sites ou páginas da web. Na propriedade href utilizamos a URL para a referência que desejamos

Sintaxe:

<a href="URL"> Conteúdo Referenciado</a>

Hiperlinks internos

São links que apontam a um lugar diferente dentro da mesma página e possibilitam acessar diferentes parágrafos ou seções quando o documento é extensos

Sintaxe:

<a href="#referencia"> Conteúdo Referenciado</a>
<a name="referencia"></a>

Hiperlinks locais

São links que apontam para outras páginas dentro de um mesmo site.

Sintaxe:

<a href="arquivo.html"> Conteúdo Referenciado</a>

Hiperlinks em imagens

É possível realizar hiperlinks em imagens envolvendo a tag de imagem com a tag de hiperlink

Sintaxe:

<a href="referencia"> <img src="imagem.jpg"> </a>

Hiperlinks para downloads de arquivos

São links que apontam para arquivos e possibilitam realizar o download desses arquivos.

Sintaxe:

<a href="arquivo.extensão" download="arquivo.extensão" type="tipo"> Conteúdo </a>

Hiperlinks para endereços eletrônicos

São links que apontam para um endereço eletrônico.

Sintaxe:

<a href="mailto:email@gmail.com"> email@gmail.com <img src="imagem.jpg"> </a>
