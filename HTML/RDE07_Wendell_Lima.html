Branches e Hospedagem

O que são Branches?

São ramificações de um projeto de software ou documento, tendo como objetivo, o desenvolvimento pararelo de:

  1. Novas funcionalidades;
  2. Correção de problemas;
  3. Alteração de código.

Tipos de Branches

Branch Master ou Main

Braço principal de um projeto em um repositório.

Branch Secundária

Braço secundário de um projeto em um repositório.
Permite fazer alterações sem afetar a principal.

Merge

Junção/Mesclagem entre os branches.

Hospedagem

Na maioria dos casos, para conseguir hospedar um site em um servidor, é necessário pagar uma certa quantia, porém, existe uma ferramenta chamada GitHub Pages, que oferem hospedagem gratuita.
É preciso apenas criar uma conta e um repositório público, com a URL:
https://nomeusuário.github.io/repositório

Passo a passo

Passo 1:
Ir no repositório escolhido;


Passo 2:
Clicar em configurações;


Passo 3:
Em configurações, clique na opção "Pages";


Passo 4:
Escolha a main principal e salve.
