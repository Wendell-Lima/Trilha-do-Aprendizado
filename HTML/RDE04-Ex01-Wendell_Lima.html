Direitos de Uso

É preciso ter cuidado na hora de pegar uma imagem na internet, normalmente as imagens possuem direitos autorais, por isso é necessário usar imagens com Licenças Creative Commons

Tipos e Editores de imagens

Existem vários tipos de extensões para imagens, assim como editores que nos ajudam a redimensionar ou a converte-lás. (Photoshop ou Gimp)

Tag para imagem

Para colocar uma imagem em HTML, é necessário usar a seguinte tag: <img src="caminho da imagem" alt="descrição para pessoas com deficiência visual">

Tag para icone

Para colocar uma imagem em HTML, é necessário usar a seguinte tag: <link rel="icon" href="diretório do icone" type="image/x-icon">

Tags para Listas

  • Para uma lista não ordenada usa-se <ul> e </ul>

  • Para uma lista ordenada usa-se <ol> e </ol

  • Para uma lista ordenada usa-se <dl> e </dl>
