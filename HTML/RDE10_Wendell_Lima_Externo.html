Cascading Style Sheets (CSS)

Definição

Folhas de Estilo em Cascata - possui como objetivo realizar a estilização de conteúdo de documentos HTML. Podem definir cores, fontes, espaçamento dentre outros.

Componentes

  • Seletores - seleciona elementos HTML com base no nome do elemento ao qual o estilo será aplicado.

  • Propriedades - o aspecto específico do elemento que você quer usar como estilo.

  • Valores - a configuração aplicada ao atributo.

Sintaxe:

seletor{
ㅤㅤpropriedades: valor; /*comentário*/
ㅤㅤpropriedades-subpropriedades: valor;
}


p{
ㅤ color:blue;
ㅤ text-align: center;
}

Formas de Utilizar CSS

Existe uma ordem seguida pelos navegadores:

INLINE => INCORPORADO => EXTERNO.

Formas de Utilizar CSS
Inline Os códigos CSS são definidos dentro de uma tag HTML.
Afetam somente a tag em que foi definida.
Incorporado Os códigos CSS são definidos no cabeçalho, dentro de uma tag head.
Afetam somente o documento em que foi definido.
Externo Os códigos CSS são definidos em um arquivo externo.
Afetam todos os documentos que estão linkados ao CSS.
