Table

Tabelas nos permitem organizar dados em linhas e colunas (uma ótima opção para organização).
Possui uma estrutura parecida com a de listas.
Tag principal usada para definir tabelas: <table>
Agora para definir os elementos que estão dentro de uma tabela usamos as seguintes tags:

<tr> --> define uma linha da tabela;
<th> --> define um cabeçalho da tabela;
<td> --> define uma célula da tabela;

Exemplo de tabela:

Meses Gastos
Janeiro R$:100
Fevereiro R$:300
Março R$:900
Abril R$:400

Del e Ins

A tag <del> permite mostrar que há uma frase/palavra que foi retirada do texto por meio do tachamento.
Já a tag <ins> permite mostrar que há uma frase/palavra que foi inserida no texto por meio do sublinhado.

Por exemplo na frase: "Eu comprei duas maças".
Irei retirar a palavra duas e trocar por cinco.
"Eu comprei duas cinco maças"

Pode ser usado também como formatador de texto, para tachar e sublinhar.

Sub e Sup

Usamos a tag <sub> para definir um texto subscrito.
E para um texto sobrescrito, usamos a tag <sup>.
São bem úteis para fórmulas matemáticas e químicas.

Exemplo de texto subscrito: "Este texto ficara subscrito agora".

Exemplo de texto sobrescrito: "Fórmula de Pitagoras: "a2=b2+c2"

Blockquote e Q

A tag <blockquote> é usada para fazer citações de outras fontes.
Já a tag <q> é usada para fazer uma curta citação.

Por exemplo, uma longa citação do Bob Marley:

Difícil não é lutar por aquilo que se quer, e sim desistir daquilo que se mais ama.
Eu desisti. Mas não pense que foi por não ter coragem de lutar, e sim por não ter mais condições de sofrer.
Agora uma citação curta, de uma das falas de Braum: Mamãe sempre dizia, 'Não perca! Quanto mais escura a noite... Mais brilhante as estrelas'

Abbr

A tag <abbr> define uma abreviatura ou acrônimo.
Por exemplo:
A OMS foi fundada em 7 de abril 1948.

Address

A tag <address> define as informações de contato do autor/proprietário de um documento ou artigo.

As informações de contato podem ser o endereço de e-mail, URL, endereço físico, número de telefone, mídias sociais, etc.

O texto elemento geralmente é colocado em itálico e os navegadores sempre adicionarão uma quebra de linha antes e depois do elemento.

Por exemplo:

Escrito por: Wendell Lima

Cite

A tag <cite> define o título de um trabalho artístico (um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.).
Como exemplo:

Os Comedores de Batata feito por Vicent Van Gogh.

Bdo

A tag <bdo> é usada para mudar o sentido do texto.
Por exemplo a frase: "O Papa é Pop".
Fica: "O Papa é Pop"

Code

A tag <code> é usada para mostrar um pedaço de código.
Por exemplo: definir uma variável como int a torna suscetível a apenas numeros inteiros.

Pre

O texto inserido dentro da tag <pre> é exibido em uma fonte de largura fixa, preservando os espaços e as quebras de linha.
O texto será exibido exatamente como escrito no código-fonte HTML.
Por exemplo: 

 Agora não preciso mais usar a tag <br> para pular linhas, pois essa tag faz isso por si só. Assim como todos os espaços a 
                    mais que eu der, ela vai mostrar.

Kbd

A tag <kbd> é usada para definir a tecla que deve ser pressionada no teclado para gerar certa ação.
O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

Exemplo:

Pressione Ctrl + C para copiar o texto.
Pressione Ctrl + V para colar o texto.
Pressione Ctrl + X para recortar o texto.
